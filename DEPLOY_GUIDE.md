# Руководство по развертыванию (Deployment Guide)
**Репозиторий:** [anyagixx/krot-prod](https://github.com/anyagixx/krot-prod)

Данное руководство описывает процесс развертывания серверов для AmneziaVPN Manager (Jovaniy Krot v2).
Архитектура требует аренды двух VPS:
1. **DE-Server (Заграничный, например Германия)** — выходная нода, где пользователи получают доступ в глобальный интернет.
2. **RU-Server (Россия)** — входная нода, где запускается панель управления и через которую пользователи подключаются изначально.

---

## Шаг 1: Развертывание выходной ноды (DE-Server)

1. Подключитесь по SSH к зарубежному серверу (Ubuntu 22.04 / 24.04).
2. Выполните команду для скачивания и запуска скрипта:
   ```bash
   curl -sSL https://raw.githubusercontent.com/anyagixx/krot-prod/main/scripts/install-de.sh | bash
   ```
3. После завершения установки в консоли будут выведены **IP-адрес сервера** и **Публичный ключ**. 
   > Обязательно скопируйте и сохраните их! Они понадобятся на Шаге 2.

---

## Шаг 2: Развертывание менеджера и входной ноды (RU-Server)

1. Подключитесь по SSH к российскому серверу (Ubuntu 22.04 / 24.04).
2. Замените переменные `ADMIN_USER`, `ADMIN_PASS` своими значениями. 
3. Заполните `DE_SERVER_IP` и `DE_SERVER_PUBKEY` данными из Шага 1.
4. Выполните установку:
   ```bash
   export ADMIN_USER="your_admin_login"
   export ADMIN_PASS="your_secure_password"
   export DE_SERVER_IP="ip_из_шага_1"
   export DE_SERVER_PUBKEY="pubkey_из_шага_1"

   curl -sSL https://raw.githubusercontent.com/anyagixx/krot-prod/main/scripts/install-ru.sh | bash
   ```

5. Скрипт развернет панель управления, настроит AmneziaWG, установит туннель к DE-серверу и скачает списки российских IP-адресов.

---

## Шаг 3: Использование панели управления

После завершения установки панель будет доступна по адресу:
`https://<IP_вашего_RU_сервера>:8080`

Скрипт `install-ru.sh` выпускает self-signed сертификат, поэтому при первом входе браузер покажет стандартное предупреждение безопасности.

1. Зайдите в панель, используя логин и пароль, заданные на Шаге 2.
2. В интерфейсе вы сможете:
   - Добавлять и удалять клиентов (генерировать конфигурации/QR-коды).
   - Следить за ресурсами сервера (CPU, RAM, Диск).
   - "На лету" менять параметры **Обфускации** VPN для всех клиентов панели в разделе настроек.
   - Изменять пароль администратора в панели управления.

> **Важно:** Списки российских IP адресов будут автоматически обновляться раз в сутки, обеспечивая бесперебойную работу локальных сайтов без туннелирования за рубеж.
